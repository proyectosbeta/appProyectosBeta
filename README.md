# Description

The app for Proyectos Beta (proyectosbeta.net)

# Tecnologies

-   React 17.0.XX
-   React native 0.68.XX
-   NodeJS v14.19.XX
-   NPM 8.13.XX
-   OpenJDK 17.0.XX
-   Gradle 7.3.XX
-   Emulator Android Pixel_XL_API_29(AVD)

# Development

## Run

```bash
npm install
npx react-native start
```

## Compile and run

```bash
npx react-native run-android
```
